# Hotel Rating System - Configuration Repository

This repository contains the centralized configuration files used by the Hotel Rating System Microservices project. The Spring Cloud Config Server fetches the configuration files from this repository and provides them to the microservices during application startup.

## Configuration Files
- application.yml (Common configuration shared across all microservices)
- user-service.yml
- hotel-service.yml
- rating-service.yml
- api-gateway.yml
